# 📧 سیستم مدیریت نامه

سیستم مدیریت نامه یک برنامه تحت وب است که با استفاده از **Python، Flask و PostgreSQL** توسعه داده شده است. این سامانه امکان ارسال، دریافت و مدیریت نامه‌ها را در محیطی ساده و امن فراهم می‌کند.

---

# 📌 معرفی پروژه

این پروژه با هدف پیاده‌سازی یک سامانه مدیریت نامه طراحی شده است که کاربران پس از ورود به سیستم می‌توانند:

- ورود امن به سامانه
- ارسال نامه به سایر کاربران
- دریافت نامه‌ها
- مشاهده صندوق ورودی
- مشاهده جزئیات هر نامه
- بارگذاری فایل پیوست
- دانلود فایل‌های پیوست
- خروج امن از حساب کاربری

# 🚀 نحوه اجرای پروژه

ابتدا پروژه را Clone کنید:

```bash
git clone https://github.com/amirkh23320-cloud/MailProject.git
```

وارد پوشه پروژه شوید:

```bash
cd MailProject
```

کتابخانه‌های موردنیاز را نصب کنید:

```bash
pip install -r requirements.txt
```

سپس پروژه را اجرا کنید:

```bash
python app.py
```

---

# 🐳 اجرای پروژه با Docker

برای اجرای پروژه توسط Docker دستور زیر را اجرا کنید:

```bash
docker compose up --build
```
